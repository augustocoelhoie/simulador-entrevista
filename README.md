Simulador de Entrevista Técnica – Dev C# Jr
Sobre o Projeto
Este projeto registra uma simulação completa de entrevista técnica para a vaga de Desenvolvedor C# Júnior.
O objetivo é demonstrar conhecimento técnico, capacidade de raciocínio e habilidades comportamentais relevantes para o cargo.
O conteúdo serve como entrega de portfólio, evidência de estudo e preparação para o mercado.
Conteúdo da Entrevista
A entrevista cobre os principais tópicos exigidos de um desenvolvedor júnior:

C# (fundamentos)

O que são namespaces
Diferença entre classes e structs
Organização e boas práticas


Entity Framework Core

O que é o DbContext
Como funciona o tracking
Como ocorre o CRUD através do EF Core


SQL

Diferença entre INNER JOIN, LEFT JOIN e RIGHT JOIN
Como e quando cada um é utilizado


Debug e raciocínio lógico

Estratégia para investigar erros intermitentes
Observabilidade, logs e hipóteses técnicas
Análise por camadas


Soft Skills


Comunicação clara
Transparência
Proatividade na resolução de bloqueios e riscos
